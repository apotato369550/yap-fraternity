# Yap Fraternity Style Guide

## Typography

**Font Family:** Noto Serif (Google Fonts)
Fallbacks: Georgia, serif

| Element | Weight | Usage |
|---------|--------|-------|
| Main heading (h1) | 700 (bold) | "YAP" hero |
| Subheading (h2, h3) | 600 (semibold) | Section titles, subtitles |
| Secondary text | 500 (medium) | "EST. 1970", labels |
| Body | 400 (regular) | Paragraphs, default text |

**Base font-size:** 21px
**Max content width:** 70vw
